# Generics Practice: Universal Record Storage 🚀

Учебный проект по глубокому изучению **Java Generics**, реализованный в рамках перехода в профессию Android-разработчика.

## 🎯 Цели проекта
Закрепление фундаментальных навыков работы с обобщениями в Java:
* **Bounded Type Parameters:** Ограничение типов (`T extends BaseRecord`).
* **Wildcards:** Использование гибких типов данных (`? extends BaseRecord`).
* **Инкапсуляция:** Работа с приватными полями и доступом через геттеры.
* **Наследование:** Корректная передача параметров через `super()`.

## 🏗️ Архитектура
Проект имитирует простую систему хранения данных (аналог CRM или складского учета):
1.  `BaseRecord` — абстрактный корень иерархии с уникальным `id`.
2.  `Product` & `SupportTicket` — конкретные сущности с уникальными полями.
3.  `RecordStorage<T>` — универсальный контейнер для безопасного хранения и поиска записей.
4.  `StorageUtils` — утилитарный класс для демонстрации работы с Wildcards.
